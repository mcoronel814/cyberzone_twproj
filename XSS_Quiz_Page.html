Introduction to Cross Site Scripting Quiz

1. What is cross-site scripting?

Malicious code that is executed to redirect users to malicious websites to steal personal info.
Injection code used to steal information from databases.
A computer program that is designed to spread across devices.
An online scam that targets users through e-mail.

The correct answer is malicious code that is executed to redirect users to malicious websites to steal personal info.

XSS is an attack where malicious code is injected into a website and is used to harm the victim’s browser. The purpose of this attack is to steal the victim’s identity data.

2. What are the two main languages used to perform XSS?

VBScript and ActiveX
Python and CSS
Flash and C#
HTML and JavaScript

HTML and JavaScript

JavaScript, HTML, VBScript, and Flash are all used to perform XSS. Javascript and HTML are the most commonly used languages to perform XSS.

3. What are the types of XSS?

Mirrored XSS, IBM-based XSS, Scripted XSS
Flash XSS, Super XSS, Storm XSS
Stored XSS, Reflective XSS, DOM-Based XSS
Static XSS, Dynamic XSS, Restore XSS

The correct answer is Static XSS, Dynamic XSS, Restore XSS

Reflective XSS occurs when the results of a malicious script are being returned to the website. Stored XSS is when the malicious script is being saved on the webserver. DOM-based attacks are also called client-side XSS.

4. Which type of XSS attack is also known as client-side XSS?

Stactic XSS
DOM-based XSS
Stored XSS
Reflective XSS

The correct answer is DOM-based XSS

DOM-based attacks are also called client-side XSS.

5. Stored XSS is when...

The malicious script is being saved on the webserver.
The scripts are used to modify the user’s browser.
The results of a malicious script are being returned to the website.
The scripts are added to the victim's profile.

The correct answer is when the malicious script is being saved on the webserver

Stored XSS is when the malicious script is being saved on the webserver. This affects users on a larger scale because its able to to attack multiple users at the same time.
