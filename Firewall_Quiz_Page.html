Introduction to Firewalls Quiz

1. Dynamic Packet Filtering Firewall is also known as..

Seal Firewall
Stateful Inspection Firewall
Proxy Firewall
Packet Filtering Firewall

The correct answer is Stateful Inspection Firewall

Stateful Inspection Firewall is also known as Dynamic Packet Filtering Firewall. This firewall is a variation of Packet Filtering Firewall.

2. What are the three major Transport Layer protocols?

TCP, ICMP, AND UDM
UDP, IMDB, CBS
ICMP, TCP, UDP
TCP, ABC, DIS

The correct answer is TCP, UDP, and ICMP

The three major Transport Layer protocols are TCP, UDP, and ICMP.

3. Which type of firewall prevents the direct connection between either end of the firewall?

Proxy Firewall
Space Defiance Firewall
State Inspection Firewall
Packet Filtering Firewall

The correct answer is Proxy Firewall

Proxy Firewall prevents the direct connection between either end of the firewall.

4. Which of the following actions stops traffic but also sends a reply message?

Accept
Push
Drop
Reject

The correct answer is reject

When traffic is rejected, it's denied access to the connect and is sent a reply message.

5. Which of the following Transport Layer protocols uses type codes?

ICMP
UDP
TCP
DIS

The correct answer is ICMP

ICMP uses type codes.
